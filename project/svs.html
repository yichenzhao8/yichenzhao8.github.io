Upload + Async Analysis

When uploading files to the server, I build an upload progress monitor using XHR on progress and unload methods for a large file that cannot finish uploading immediately. Once the upload is complete, a FormData object which appended audio file will be sent to Upload API and Analysis API for analysis.

Switch Language

I created two JSON files that are identical in structure but different in translation. I also design buttons that represent each language. Once the language target has clicked, a function will read that language JSON file, search for the key which matches the data-attribute in HTML and replace the laguage in the html. In addition, cookie will be set in order to remain that same language if the user close the browser or visit the site again.
